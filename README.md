# Tariff-Page
Вёрстка страницы тарифов для «Газпромбанк Мобайл»

# Содержание

1. [О проекте](#project)
2. [Запуск проекта](#start-project)

## <a name="project">О проекте</a>

В рамках проекта создана страница `tariffs.html` с двумя основными разделами:
* «Выбор тарифа» — сетка карточек с вариантами тарифов;
* «Заказать сим‑карту» — форма с пошаговым оформлением заказа.

Проект выполнен для демонстрации навыков в портфолио: вёрстка по макету Figma (Pixel Perfect), БЭМ, Flexbox/Grid, спрайт, «липкий» подвал.

## <a name="start-project">Запуск проекта</a>

1. На странице репозитория нажмите кнопку `Code` (зелёная кнопка вверху справа).
2. В выпадающем меню выберите `Download ZIP`.
3. Распакуйте архив в удобное для вас место на компьютере.
4. Установите приложение **Visual Studio Code**.
5. Установите расширение **Live Server** для VS Code.
6. Откройте папку проекта в VS Code.
7. Кликните правой кнопкой мыши по файлу `tariffs.html` и выберите **Open with Live Server**.
8. Браузер откроется автоматически по адресу `http://127.0.0.1:5500/` (или аналогичному).
